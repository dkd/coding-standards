h1. PHP Coding Standards by dkd

This repository contains php sniffs for TYPO3, based on the "Pear package PHP_CodeSniffer":http://pear.php.net/package/PHP_CodeSniffer/.

It also delivers a ruleset for "PHPMD - PHP Mess Detector":http://phpmd.org/.
